# Bossearch's Nvim-Config

This is a rewrite from my nixvim config but with some improvement. i'm using Neovim's builtin plugin manager `vim.pack`, so it only works on Neovim > v.0.12.0. And i use [lz.n](https://github.com/lumen-oss/lz.n) for lazy loading plugins keeping my nvim startuptime fast as possible. and it have some custom 'plugin' that work with my particular workflow.

I used to configure Neovim using [Nixvim](https://github.com/nix-community/nixvim), but rebuilding every single time i changed something became tedious. nvim is where i spend almost all my time coding and taking notes, so i constantly update my config. Don't get me wrong, Nixvim is great and the contributors are awesome—you can do almost anything you want in vanilla lua. But after hundreds of times, the rebuilding process just got old. That's why i'm separating my nvim config into this repo.
